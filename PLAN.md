## 💰 광고 전략 (AdMob)

### 광고 유형
- **인터스티셜 (전면 광고):** 특정 액션 완료 후 (앱별 최적 타이밍)
- **배너 광고:** 각 페이지 하단 고정 (일부 앱은 상단)

### 앱별 인터스티셜 배치

| 앱 | 인터스티셜 타이밍 |
|----|-----------------|
| 습관 추적기 | 습관 체크 완료 후 |
| 가계부 | 월간 리포트 열 때 |
| 일기/저널 | 저장 완료 후 |
| 운동 세트 기록기 | 운동 세션 종료 후 |
| 단위 변환기 | N번째 변환마다 |
| 대출 이자 계산기 | N번째 계산마다 |
| 백색소음/수면 사운드 | 세션 종료 시 |
| 응급처치 가이드 | ⚠️ 인터스티셜 없음 (긴급 앱) |
| 튜너 | 세션 종료 시 |
| 독서 기록 앱 | 책 추가/완독 기록 후 |

### 배너 위치
- 기본: 화면 **하단** 고정
- 예외: 백색소음, 튜너 → 컨트롤 버튼이 하단일 경우 **상단**으로

## 🔧 기술 스택

| 역할 | 선택 |
|------|------|
| 앱 프레임워크 | Flutter (Dart) |
| 모노레포 관리 | `melos` |
| Flutter 버전 고정 | `fvm` + `.fvmrc` |
| 광고 | `google_mobile_ads` |
| 상태관리 | `riverpod` |
| 단순 저장 | `shared_preferences` |
| 복잡 데이터 | `isar` (앱별 선택적 적용) |
| 오디오 (백색소음/튜너) | `just_audio` / `flutter_sound` |
| 소개 페이지 | HTML/CSS (GitHub Pages, `docs/` 폴더) |
| 패키지 업데이트 | `renovate` PR 자동화 |
